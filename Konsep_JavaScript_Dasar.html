BAB-1 Mengenal JavaScript

Pengertian JavaScript

JavaScript itu termasuk dalam client side programming language, artinya dapat diproses di dalam web browser dengan mudah sama seperti HTML dan CSS. Tetapi, karena perkembangannya yang terus berlanjut, JavaScript ini juga dapat digolongkan dalam server side programming language. JavaScript yang berjalan di sisi server ini adalah node.js.

JavaScript ini termasuk ke dalam bahasa pemrograman tingkat tinggi (mirip dengan bahasa inggris sehari-hari).

JavaScript itu salah satu bahasa pemrograman yang tidak bertipe, artinya kita tidak perlu menetapkan sebuah variabel harus bertipe data tertentu.

JavaScript dikembangkan oleh ECMA (European Computer manufacturers Association), yaitu lembaga standarisasi eropa khusus komputer.

Bahasa pemrograman JavaScript ini, source codenya juga dapat dilihat langsung di halaman website seperti HTML dan CSS, yaitu dengan klik kanan pada website nya dan pilih "View Page Source".

Secara sederhana JavaScript digunakan untuk memanipulasi HTML sehingga halaman website menjadi lebih interaktif.

HTML, CSS, dan JavaScript itu secara bersama-sama dapat membangun sebuah website yang sangat dinamis. HTML bertugas untuk struktur dasar dan isi dari halaman, CSS untuk mempercantik tampilan, dan JavaScript untuk menangani bagian interaksi nya.

JavaScript dan JAVA

Bahasa pemrogrman JavaScript dan JAVA itu secara penamaan ada kemiripan, tetapi keduanya ini sepenuhnya berbeda. Penamaan ini ada kemiripan karena JavaScript ingin memiliki kepopuleran yang sama seperti bahasa pemrograman JAVA waktu itu. Ini adalah ide marketing dari Netscape Commnication (perusahaan yang pertama kali mengembangkan JavaScript). Secara penulisan sintax antara JavaScript dan Java itu ada kemiripan juga karena keduanya sama-sama berinduk dari bahasa pemrograman C dan C++.

JavaScript dan DOM

JavaScript dan DOM itu itu sepenuhnya berbeda. JavaScript adalah bahasa pemrograman dan dikemangkan oleh ECMA sedangkan DOM adalah konsep struktur HTML yang dikembangkan oleh W3C.

Berbicara mengenai DOM, ada informasi dasar yang perlu diketahui yaitu tentang "Window Object".

Window Object itu memiliki beberapa object:

  1. location object
  2. history object
  3. document object
    • HTML (element node).
  4. navigator object
  5. screen object
